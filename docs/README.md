# BuildPaw GitHub Pages

This folder contains the static site for [BuildPaw](https://github.com/birhos/BuildPaw) GitHub Pages.

## Deployment

1. Push the `docs/` folder to your repository
2. Go to **Settings → Pages** in your GitHub repo
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: main (or master)
   - **Folder**: /docs
4. Save — GitHub will deploy the site to `https://<username>.github.io/BuildPaw/`
